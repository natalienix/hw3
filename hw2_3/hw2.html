Unix Shell Command Name Description Example
ls

Invisible text

 lists files 
$ ls
hw2.html
README.md
tree shows begining of directory 
$ tree
.
|-- config.dat
|-- data
|   |-- data1.bin
|   |-- data2.sql
|   |-- data3.inf
|-- images
|   |-- background.jpg
|   |-- icon.gif
|   |-- logo.jpg
|-- program.exe
|-- readme.txt
file tells what kind of file it is 
$ file file.c
file.c: C program text
type what type of command 
$ type test
test is a shell builtin
apropos used to search man page files - like a mini google 
$ apropos mount
free (1)- Display amount of free and used memory in the system
mklost+found (8)- create a lost+found directory on a mounted Linux second extended file system
mount (8)- mount a file system
mountpoint (1)- see if a directory is a mountpoint
ntfsmount (8)- Read/Write userspace NTFS driver.
sleep (1)- delay for a specified amount of time
switch_root (8)- switch to another filesystem as the root of the mount tree.
umount (8)- unmount file systems
pwd prints working directory 
$ pwd
/usr/local/bin
cd changes directory 
$ cd games
usr:~/games$
mkdir makes an empty directory 
$ mkdir -p /tmp/a/b/c
echo turns list of arguments to text 
$ echo Hello world
Hello world
cat appends one thing to another 
$ cat file1.txt file2.txt > newcombinedfile.txt
less puts you into a different place where you can view everything on one screen 
$ less -M readme.txt
